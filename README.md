# CL-Andro - Release Repository

This repository contains the build workflow scripts, license, and instructions to download and install **CL-Andro** — a powerful terminal emulator application for Android.

---

## 📥 How to Download & Install

1. Navigate to the **Releases** section on the right side of this GitHub repository page.
2. Download the latest `cl-andro-release.apk`.
3. Transfer the APK to your phone and install it (make sure you allow installation from "Unknown Sources" in your browser/file explorer if prompted).

---

## 📦 About CL-Andro

**CL-Andro** (`com.zk.clandro`) is a feature-rich terminal emulator for Android. It provides a Linux environment with package management, allowing you to run command-line tools and utilities directly on your Android device.

### Key Features
- Full Linux terminal environment on Android
- Package management via `apt`
- Extensible with plugins and add-ons
- Integration with Cluster Family ecosystem (Cluster Files, Cluster Auto, CL-Browser)

---

## 🔒 Authorization with Cluster Auto

CL-Andro is **automatically pre-authorized** to connect to the Cluster Auto daemon for elevated file system access. Once Cluster Auto's daemon is running, CL-Andro can access root-level directories without additional configuration.
